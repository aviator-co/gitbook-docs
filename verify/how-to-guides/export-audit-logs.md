# Audit export

Audit export is on the roadmap and is not currently available.

### What's available today

Verification records are visible in two places:

* **Dashboard** — each runbook's verify page lists past verification runs, criteria, and verdicts.
* **GitHub check** — the `aviator/verify` check on each PR includes a markdown summary linking back to the dashboard.

There is no built-in export to JSON, CSV, or PDF, and no scheduled report generation. If you need verification history for compliance reporting, contact [support@aviator.co](mailto:support@aviator.co) — we are actively working with design partners on what audit-grade export should produce.
